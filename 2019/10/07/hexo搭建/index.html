hexo博客的搭建

​ 简单的介绍一下hexo是怎么搭建的，以免下次又忘了。

​ 先完成hexo安装的操作：

​ 先下载好git bash和nodejs的npm管理工具后，就可以很简单的下载hexo啦。

  • 先创建一个文件夹（用来存放所有blog的东西），然后cd到该文件夹下。

  • 安装hexo命令：npm i -g hexo

然后初始化一下，新建一个文件（名字随便取），打开git bash，初始化命令：hexo init

然后修改主题什么的到github上找一个喜欢的，在config文件里面配置一下,最后就可以上传到gitHub上啦。

上传到github之前，还要安装一个东西：

npm install hexo-deployer-git --save（这样才能将你写好的文章部署到github服务器上并让别人浏览到）

然后执行以下命令 hexo g （生成文件） hexo s (开启本地服务器) hexo d(上传到github)
